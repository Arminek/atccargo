Projekt strony ATC-Cargo
=======

### System rang
* Szef
* V-CE Szef
* Dyspozytor
* Kierowca
* Okres testowy

### Uprawnienia
Niektóre rangi mają możliwość przeglądania transportów, przeglądania użytkowników.
| Zakładka                | Ranga             |
| ----------------------- | ----------------: |
| Przeglądaj transporty   | Szef-Dyspozytor   |
| Przeglądaj użytkowników | Szef              |
| Dodaj pracownika        | Szef              |
